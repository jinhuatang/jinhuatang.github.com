Markdown常用语法备忘
======================

标题`H1` `H2` `H3` `H4` `H5` `H6`
----------------------
#H1`#H1`
```
H1
=====```
##H2`##H2`
```
H2
-------```
###H3`###H3`
####H4`####H4`
#####H5`#####H5`
######H6`######H6`

段内容`p`
----------------------
普通空行开始即可

引用`blockquote`
----------------------
`>`
> 单层引用

`>>`
>> 多层引用

列表`ul` `ol`
----------------------
无序列表`*` `+` `-`

* a
* b
* c

有序列表`1.` `2.` `3.`

1. a
2. b
3. c

链接与图片
--------------------
链接`[链接文字](url "title")`  
自动链接`<url>`  
图片`![图片文字](url "title")`  
图片加链接`[![图片文字](url "title")](url "title")`

换行
-------------------
行尾多于一个空格就可以产生`<br/>`

强调
-------------------
`em`为`*`或者`_`  
*em*  
`strong`为`**`或者`__`  
__strong__
