shell parameters 2013-02-20

虽然shell很强大，但是我一直都没有认真学过和写过，最近在折腾下小东西，需要写点shell来把自己从repeat中释放出来，首先碰到问题就是shell如何接受参数，只有这样才能进行多样性的input和output。

这里记下以作备忘：

  • $#：这里的#代表number，自然数表示第几个参数，第0个指参数前的命令
  • $*：代表所有参数

示例代码：

#!/bin/bash
#test.sh
echo all params: $*
echo first param: $1

运行：

./test.sh p1 p2 p3
all params: p1 p2 p3
first param: p1
